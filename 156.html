DEF
This is speciﬁed in the grammar as

name
I need this, so I match it with

LPAREN
I assumed I received a
def

match(tokens, 'LPAREN')

params
In the BNF I deﬁne
params
as a new “grammar production” or “grammar rule.” That
means in my Python code I need a new function. In this function, I can call that function
with
params = parameters(tokens)
. Later I deﬁne the
parameters

comma-separated parameters for functions.
RPAREN
I discard this but require it again with
match(tokens, 'RPAREN')
.
COLON
Once again, I discard the match
match(tokens, 'COLON')

body
I’m actually skipping the body here because Python’s indented syntax is a little too hard
for this simple example code. You will need to handle this problem in the exercise unless you

That is basically how you read an ABNF speciﬁcation and systematically translate it into code. You start
at the root, implement each grammar production as a function, and leave the scanner to handle the
simple tokens (which I denote with CAPITAL letters).

The ﬁrst step to doing this is scanning the text looking for “tokens.” At the scanning phase a language
like Python doesn’t ﬁrst care what’s a symbol (

convert the input language into patterns of text called “tokens.” It does this by applying a sequence of

from Exercise 31 that a regular expression is a way to tell Python what sequences of characters to
match or accept. All the Python interpreter does is use many regular expressions to match every token
it understands.

Finally, the best way to speed anything up is to not do it at all. If you’re doing multiple checks of the
same condition, ﬁnd a way to avoid doing it more than once. If you’re calculating the same column in a
database repeatedly, do it once. If you call a function in a tight loop, but the data rarely changes, check
out memoization or pre-calculate a table. In many cases you can trade storage for computation by simply
calculating things ahead of time and storing them once.

Exercise Challenge
The
cat
command is short for “concatenate,” and it’s most commonly used to dump the contents of a
ﬁle to the screen. You use it like this:
cat somefile.txt
This command will spit out the contents of
somefile.txt
. That’s not actually its original purpose.
Originally it was for combining more than one ﬁle—thus it was called
cat
. To do that simply add each
ﬁle to
cat
:
cat A.txt B.txt C.txt
The
cat
command will then go through each ﬁle, write its contents out, and then exit when it has gone
through all of them. The problem is, how does this possibly concatenate ﬁles? To do that you’ll also need
to use the POSIX ﬁle redirection features found in your Terminal:
cat A.txt B.txt C.txt > D.txt
